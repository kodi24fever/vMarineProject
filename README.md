# React Router Mini Project

#### Make sure you have node version 8
```sh
$ nvm install 8
$ nvm use 8
$ node -v
```

##### Install packages:
```
$ npm install
```

###### We'll use Webpack server to simulate the final server

```sh
$ npm run c9
```

##### Now code and save. No need to ```sh $ npm run build ```

...Magic?

Yes.
